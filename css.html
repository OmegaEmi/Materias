Css

Css o Cascading Style Sheets o hojas de estilo en cascada determinan la forma en la que se verán los elementos html

Se puede usar css de tres maneras:

En linea:Usando el atributo style en los elementos html

interno:Usando un elemento style en la sección head de la página

Externo:Usando un archivo css externo
